Violin plot with ggplot2

Jiaqiang Zhu, 02/18/2020

R code for violin plot with points using ggplot2: 

library(ggplot2)
set.seed(1)
b <- rpois(1000,lambda=2)
pltdat <- data.frame(count=b)
ggplot(pltdat, aes(y = count,x="count",fill=hue_pal()(1))) +
	geom_violin() +   
	geom_jitter(height=0,size=1) +
	theme_bw()+
	theme(legend.position="none")
