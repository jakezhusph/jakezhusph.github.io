Extract Legend from ggplot

Jiaqiang Zhu, 02/25/2020

R code for extracting Legend from ggplot: 

library(ggplot2)
g_legend <- function(a.gplot){ 
tmp <- ggplot_gtable(ggplot_build(a.gplot)) 
leg <- which(sapply(tmp$grobs, function(x) x$name) == "guide-box") 
legend <- tmp$grobs[[leg]] 
return(legend)} 

## draw the plot
library(grid)
grid.newpage()
grid.draw(g_legend(g2)) 

## can save with ggsave
g3 <- g_legend(g2)
